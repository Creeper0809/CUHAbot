# 스킬 시스템 (Skill System) - 덱빌딩 방식

## 개요

스킬은 **10개의 슬롯**에 장착하여 덱을 구성합니다. 매 턴 덱에서 **랜덤으로 1개의 스킬이 자동 발동**됩니다. 같은 스킬을 여러 슬롯에 장착하면 해당 스킬의 발동 확률이 증가합니다.

---

## 덱 시스템 기본 규칙

### 슬롯 규칙
| 규칙 | 설명 |
|------|------|
| 총 슬롯 | 10개 (고정) |
| 중복 장착 | **허용** (같은 스킬 최대 10개) |
| 빈 슬롯 | 불가 (모든 슬롯 필수 장착) |
| 발동 방식 | 매 턴 10개 중 랜덤 1개 발동 |

### 발동 확률 계산
```
스킬 발동 확률 = (해당 스킬 장착 수 / 10) × 100%

예시: 화염구 3개, 힐 2개, 강타 5개
- 화염구: 30%
- 힐: 20%
- 강타: 50%
```

### 덱 구성 예시
```
┌─────────────────────────────────────────────────────────────┐
│                  버서커 덱 (공격 집중)                        │
├────┬────┬────┬────┬────┬────┬────┬────┬────┬────┤
│강타│강타│강타│화염│화염│분노│분노│흡혈│치유│치유│
│×3 │    │    │×2 │    │×2 │    │×1 │×2 │    │
└────┴────┴────┴────┴────┴────┴────┴────┴────┴────┘
발동 확률: 강타 30%, 화염 20%, 분노 20%, 흡혈 10%, 치유 20%
```

## 스킬 분류

| 타입 | 설명 | 발동 시 효과 | 권장 장착 수 |
|------|------|-------------|-------------|
| attack | 데미지 스킬 | 적에게 데미지 | 4-7개 |
| heal | 회복 스킬 | HP 회복 | 1-3개 |
| buff | 강화 스킬 | 자신 버프 | 1-3개 |
| debuff | 약화 스킬 | 적 디버프 | 1-3개 |
| ultimate | 궁극기 | 강력한 효과 + 덱 소모 | 1개 |

### 궁극기 특수 규칙
- 궁극기 발동 시 해당 **슬롯이 전투 끝까지 비활성화**
- 대신 매우 강력한 효과
- 10% 확률 (1슬롯)이지만 발동 시 결정적

### 패시브 vs 액티브 트레이드오프

| 구성 | 장점 | 단점 |
|------|------|------|
| 패시브 多 (5개+) | 안정적인 상시 효과 | 발동 스킬 부족, 전투 길어짐 |
| 액티브 多 (8개+) | 강력한 스킬 발동 | 안정성 부족, 운 의존 |
| 균형형 (3:7) | 적당한 안정성 + 화력 | 특화 효과 부족 |

**핵심 전략**: 패시브는 슬롯을 차지하지만 발동하지 않으므로, 너무 많이 넣으면 전투에서 스킬 발동 확률이 낮아집니다. 하지만 파밍이나 탐험 패시브는 전투 외에서도 효과가 있으므로 상황에 맞게 조절해야 합니다.

#### 🔥 화상 폭발 덱
```
화염구×2 + 화염 폭발×2 + 불씨 키우기×2 + 화상 폭발×2 + 영원한 불꽃×1 + 응급처치×1
```

**전략**: 화상 스택을 최대한 쌓고 화상 폭발로 마무리

```
얼음 화살×3 + 동결 준비×2 + 빙결×2 + 빙결 분쇄×2 + 영구 동토×1
```
**전략**: 둔화로 동결 확률 올리고, 동결 후 분쇄
**콤보**: 얼음 화살 → 동결 준비 → 빙결 → 빙결 분쇄 (400%)

## 추천 덱 빌드

### 🔥 화염 버서커 덱
```
화염구×3 + 화염 폭발×2 + 분노×2 + 강타×2 + 응급 처치×1
```
- **전략**: 화상 스택으로 지속 피해 + 분노 버프로 폭딜
- **시너지**: 화염 5개 → 화염 데미지 +20%
- **장점**: 높은 DPS
- **단점**: 방어력 감소, 긴 전투에 불리

```
얼음 화살×2 + 빙결×3 + 눈보라×2 + 치유×2 + 수비 태세×1
```
- **전략**: 지속적인 동결/둔화로 적 행동 방해
- **시너지**: 냉기 7개 → 동결 확률 대폭 상승
- **장점**: 적 컨트롤 우수
- **단점**: 순수 딜 부족

### 💚 불멸의 탱커 덱
```
응급 처치×2 + 치유×2 + 재생×2 + 수비 태세×2 + 어둠의 갑옷×1 + 강타×1
```
- **전략**: 회복으로 버티며 천천히 처치
- **시너지**: 힐 4개 + 버프 3개 → 생존력 극대화
- **장점**: 거의 안 죽음
- **단점**: 전투 시간 김

### 🌑 흡혈 지속전 덱
```
생명력 흡수×3 + 어둠의 갑옷×2 + 저주×2 + 강타×2 + 영혼 수확×1
```
- **전략**: 흡혈로 생존하며 스노우볼
- **시너지**: 암흑 5개 → 흡혈 +15%
- **장점**: 회복 아이템 필요 없음, 장기전 강함
- **단점**: 초반 다소 약함

### ⚡ 올인 버스트 덱
```
강타×3 + 일섬×2 + 파워 스트라이크×2 + 분노×2 + 최후의 일격(궁)×1
```

```
빛의 화살×2 + 치유의 빛×2 + 생명력 흡수×2 + 저주×2 + 축복×1 + 어둠의 갑옷×1
```

## 초월 스킬 시스템 (Lv.85+ 해금)

### 초월 스킬이란?
- 각성 스킬의 **최종 진화**
- 초월자의 영역 클리어 후 해금
- 각성 스킬 + 초월 재료로 업그레이드

### 초월 조건
1. 해당 스킬 각성 완료
2. 초월의 정수 10개
3. 초월자의 영역 클리어

### 초월 스킬 특수 효과
- 슬롯 소모 없이 궁극기급 효과
- 발동 시 확정 추가 효과
- 같은 스킬 연속 발동 페널티 무시

## DB 연동

```sql
-- User_Skill_Deck 테이블 (덱 구성)
CREATE TABLE User_Skill_Deck (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id VARCHAR(50) NOT NULL,
  slot_index INT NOT NULL,  -- 0-9
  skill_id INT NOT NULL,
  FOREIGN KEY (user_id) REFERENCES UserSpec(id),
  FOREIGN KEY (skill_id) REFERENCES Skill(id),
  UNIQUE KEY (user_id, slot_index)
);

-- User_Passive_Skill 테이블 (패시브)
CREATE TABLE User_Passive_Skill (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id VARCHAR(50) NOT NULL,
  slot_index INT NOT NULL,  -- 0-3
  skill_id INT NOT NULL,
  FOREIGN KEY (user_id) REFERENCES UserSpec(id),
  FOREIGN KEY (skill_id) REFERENCES Skill(id),
  UNIQUE KEY (user_id, slot_index)
);

-- Skill 테이블
CREATE TABLE Skill (
  id INT PRIMARY KEY,
  name VARCHAR(50) NOT NULL,
  description TEXT,
  type ENUM('attack', 'heal', 'buff', 'debuff', 'ultimate', 'passive'),
  element ENUM('none', 'fire', 'ice', 'lightning', 'water', 'holy', 'dark'),
  grade_id INT,
  damage_percent INT DEFAULT 100,
  effect_json JSON,  -- 부가 효과 설정
  skill_book_item_id INT,
  FOREIGN KEY (grade_id) REFERENCES Grade(id)
);
```
