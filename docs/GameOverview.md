# CUHA Bot RPG 게임 기획서

## 게임 개요

CUHA Bot은 디스코드 기반의 **덱빌딩 턴제 RPG** 던전 탐험 게임입니다. 플레이어는 **10개의 스킬 슬롯**에 스킬을 장착하고, 매 턴 **랜덤으로 발동되는 스킬**로 전투합니다. 어떤 스킬 조합을 구성하느냐가 승패를 좌우합니다.

---

### 전투 흐름

```
[턴 시작]
    │
    ▼
[스킬 덱에서 랜덤 1개 선택] ← 가중치 적용 가능
    │
    ▼
[스킬 자동 발동]
    │
    ├─→ 공격 스킬: 데미지 계산 → 적에게 피해
    ├─→ 버프 스킬: 자신에게 버프 적용
    ├─→ 힐 스킬: HP 회복
    └─→ 디버프 스킬: 적에게 디버프 적용
    │
    ▼
[몬스터 턴] ← 몬스터도 랜덤 스킬 사용
    │
    ▼
[턴 종료] → [버프/디버프 턴 감소] → [DOT 피해 적용]
    │
    ▼
[다음 턴 또는 전투 종료]
```

### ⚠️ 전투 중 제한 사항

**전투가 시작되면 아래 행동이 불가능합니다:**

| 행동 | 전투 중 | 전투 외 |
|------|--------|--------|
| 🎒 아이템 사용 | ❌ 불가 | ✅ 가능 |
| 🔄 스킬 덱 변경 | ❌ 불가 | ✅ 가능 |
| 👕 장비 변경 | ❌ 불가 | ✅ 가능 |
| 🏃 도주 | ⚠️ 일부 가능 | - |

**핵심 원칙**: 전투 전에 덱과 장비를 신중하게 구성해야 합니다. 전투 중에는 오직 랜덤하게 발동되는 스킬에 의존합니다.

```
┌─────────────────────────────────────────────────┐
│  ⚠️ 전투 시작!                                  │
│                                                 │
│  전투 중에는 아이템 사용 및 스킬 변경이         │
│  불가능합니다.                                  │
│                                                 │
│  전투 전 준비를 철저히 하세요!                  │
│                                                 │
│  현재 덱: [화염×3] [힐×2] [버프×2] [공격×3]    │
│  현재 HP: 1,500 / 1,500                        │
│  보유 포션: HP 포션 x5 (전투 중 사용 불가)     │
│                                                 │
└─────────────────────────────────────────────────┘
```

**도주 규칙:**
| 상황 | 도주 가능 여부 |
|------|---------------|
| 일반 몬스터 | ✅ 가능 (50% 성공률) |
| 엘리트 몬스터 | ❌ 불가 |
| 보스 | ❌ 불가 |
| 주간 타워 | ❌ 불가 |
| 레이드 | ❌ 불가 |

### 👥 전투 형태

전투는 다양한 형태로 발생할 수 있습니다.

#### 1:1 전투 (기본)
```
┌─────────────────────────────────────────────────┐
│                  ⚔️ 1:1 전투                    │
│                                                 │
│    [플레이어]  ←──⚔️──→  [몬스터]              │
│     HP: 1500              HP: 800               │
│                                                 │
└─────────────────────────────────────────────────┘
```

#### 1:다 전투 (다구리)
몬스터 여러 마리가 동시에 플레이어를 공격합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                  ⚔️ 1:다 전투 - 포위당했다!                         │
│                                                                     │
│                        [고블린 A]                                   │
│                         HP: 300                                     │
│                            ↓ ⚔️                                    │
│                                                                     │
│   [고블린 B] ──⚔️──→  [플레이어]  ←──⚔️── [고블린 C]              │
│    HP: 300              HP: 1500              HP: 300               │
│                                                                     │
│                            ↑ ⚔️                                    │
│                        [고블린 D]                                   │
│                         HP: 300                                     │
│                                                                     │
│  ─────────────────────────────────────────────────────────────────  │
│  ⚠️ 경고: 적 4마리가 매 턴 공격합니다!                              │
│  💡 전체 공격 스킬이 유리합니다.                                    │
└─────────────────────────────────────────────────────────────────────┘
```

**다구리 규칙:**
| 적 수 | 적 공격 방식 | 플레이어 단일 스킬 | 플레이어 전체 스킬 |
|-------|-------------|-------------------|-------------------|
| 2마리 | 2회 공격/턴 | 1마리에게 적용 | 모두에게 50% 피해 |
| 3마리 | 3회 공격/턴 | 1마리에게 적용 | 모두에게 40% 피해 |
| 4마리 | 4회 공격/턴 | 1마리에게 적용 | 모두에게 35% 피해 |
| 5마리+ | 5회 공격/턴 | 1마리에게 적용 | 모두에게 30% 피해 |

**다구리 발생 조건:**
| 상황 | 확률 | 적 수 |
|------|------|-------|
| 일반 던전 | 20% | 2-3마리 |
| 고급 던전 | 30% | 2-4마리 |
| 엘리트 던전 | 40% | 3-5마리 |
| 기습당함 | 100% | 2-4마리 |
| 보스 소환수 | 상황별 | 보스 + 1-3마리 |

### ⚔️ 전투 인카운터

#### 일반 전투
- **설명**: 던전에 서식하는 몬스터와의 전투
- **보상**: 경험치, 골드, 일반 드롭

#### 엘리트 전투
- **설명**: 강화된 몬스터와의 전투 (스탯 1.5배)
- **특징**: 특수 패턴 보유, 도주 불가
- **보상**: 경험치 2배, 고급 드롭 확률 +30%

#### 보스 조우
- **설명**: 던전의 최종 보스
- **특징**: 고유 스킬, 페이즈 전환, 도주 불가
- **보상**: 확정 레어 드롭, 던전 클리어 보상

### 거래 방식

#### 1. 경매 (Auction)
일정 시간 동안 입찰을 받아 최고 입찰자에게 판매.

| 설정 항목 | 범위 | 설명 |
|----------|------|------|
| 시작가 | 100G ~ | 최소 입찰 시작 가격 |
| 즉시구매가 | 선택사항 | 설정 시 즉시 거래 가능 |
| 경매 기간 | 1h / 6h / 12h / 24h / 48h | 경매 진행 시간 |
| 최소 입찰 단위 | 시작가의 5% | 입찰 시 최소 증가폭 |

#### 2. 즉시 판매 (Direct Sale)
고정 가격으로 즉시 판매.

