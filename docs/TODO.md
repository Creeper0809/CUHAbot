2. **전투 중 스킬 변경 불가**
3. **주간 타워: 아이템 일절 불가, 스킬은 층 사이에만 변경 가능**
4. **패시브도 10개 스킬 슬롯 안에 장착**
5. **레벨과 등급은 독립적 (저레벨 고등급 가능)**

### 의존성 체인
```
사용자 시스템 → 세션 관리 → 던전 탐험 → 전투 시스템
                                    ↓
                              스킬 덱 시스템 (Bag)
                                    ↓
                              데미지 계산
                                    ↓
                              키워드/상태이상 → 콤보 체인
                                    ↓
                              키워드 밀도 시너지
                                    ↓
                              스킬 숙련도
```
