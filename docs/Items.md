# 아이템 시스템 (Item System)

## 개요

아이템은 캐릭터를 강화하고 전투를 유리하게 만드는 핵심 시스템입니다.

---

## 아이템 분류

| 타입 | 코드 | 설명 |
|------|------|------|
| 장비 | equip | 착용하여 스탯을 올리는 아이템 |
| 소비 | consume | 사용하면 사라지는 아이템 |
| 스킬북 | skill | 사용하여 스킬을 습득하는 아이템 |
| 재료 | material | 제작/강화에 사용되는 아이템 |
